BM25-Based Range-Searchable Encryption Archetype

BM25 是一种广泛用于信息检索的打分算法，它根据关键词在文档中的频率和文档的整体长度，计算文档与查询之间的相关性得分，在这个范围可搜索加密算法中，BM25 被引入用于对密文索引打分排序，以保障数据隐私的同时提升查询效率与结果质量

BM25 算法

用于信息检索，根据词频与文档长度评估搜索内容与文档的相关性

基于配对的身份认证

利用双线性配对构造身份认证机制，保障通信安全

AES 加密

一种对称加密算法，速度快，适合加密大量数据，用于对文档 ID 进行加密

VFE-Plus 加密

可验证函数加密方案，支持功能正确性验证，用于对 BM25 矩阵索引内容进行加密

分词算法

基于 HanLP 分词库，将文档划分为有意义的单词，用于后续文本处理和索引构建

公钥加密

使用公开密钥加密关键词序列，搜索时使用私钥解密，保障机密性
