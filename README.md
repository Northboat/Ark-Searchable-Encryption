# Ark Searchable Encryption

可搜索加密仿真平台

- 基于 Pairing 的 SE 算法仿真
- 基于 BM25 算法的范围可搜索加密系统原型
- 基于属性加密的可搜索加密四叉树构建与检索
- 签名算法仿真

环境

- Win 10
- JDK 17
- JPBC 2.0.0
- IDEA 2022.3
- Maven 3.9.1
- Spring Boot 3.0.x
